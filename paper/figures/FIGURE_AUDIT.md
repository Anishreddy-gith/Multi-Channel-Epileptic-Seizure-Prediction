## Review Notes

- Use vector PDF/SVG where the venue allows vector graphics.
- Use PNG versions for raster-only submission systems.
- Captions should state what is measured and avoid implying clinical deployment.
- PR-AUC and F1 horizon figures should be interpreted with the positive prevalence table because prevalence changes with horizon length.
- The Hybrid model-comparison entry must remain marked incomplete in text whenever the figure is discussed.
